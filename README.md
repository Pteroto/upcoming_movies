# Para configurar o projeto

## SDK do Android:

sdk 28

## Configuraçãoo da API

 - Crie uma conta no www.themoviedb.org
 - Copie a API KEY (v3 auth) no settings > API após criar a conta
 - Cole no arquivo UpcomingMovies\app\src\main\java\br\com\gustavomonteiro\upcomingmovies\data\network\ConfigValues.kt na variável ApiKey
